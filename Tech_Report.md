
# 기술 보고서 (Tech Report)

## 1. 프로젝트 개요
CGV 영화 웹사이트로부터 현재 상영작들의 리뷰 데이터를 수집하고, 수집된 리뷰를 분석하여 감성 분류 결과를 시각화하고 저장하는 과정을 자동화한 것입니다.

---

## 2. 사용 기술 및 도구
- **Selenium**: 웹 페이지에서 리뷰 및 영화 정보 크롤링
- **Pandas**: 수집한 리뷰 데이터 정제 및 CSV 파일 저장
- **Matplotlib**: 감성 분석 결과를 시각화하는 데 사용
- **WordCloud**: 리뷰 텍스트에서 자주 등장하는 키워드를 시각화
- **Jupyter Notebook**: 전체 워크플로우를 문서화하고 실행

---

## 3. 구현 내용 요약

### 3.1 크롤링
- CGV 메인 영화 페이지에서 영화 정보를 가져옴
- 각 영화 페이지로 이동해 리뷰 및 작성자, 날짜를 수집
- 리뷰가 더 존재할 경우 '다음 페이지'를 반복 클릭하여 모든 리뷰를 수집

### 3.2 감성 분석
- 사전 정의된 긍정 단어 리스트 (`pos_words`)와 부정 단어 리스트 (`neg_words`)를 기준으로 간단한 룰 기반 감성 분석 수행
- 리뷰 본문 내 키워드 빈도수를 세어 긍정 / 부정 / 중립으로 분류

### 3.3 저장 및 정리
- 영화별로 리뷰 CSV 파일 저장
- 모든 리뷰를 하나의 통합 파일로 저장 (`전체_리뷰통합.csv`)
- 저장 중 중복 경로가 발생하면 파일명 뒤에 자동으로 `(1)`, `(2)` 등을 붙여 저장

### 3.4 시각화
- **영화별 감성 분석 원형 차트**: 각 영화의 긍/부정/중립 분포 시각화
- **영화 전체 리뷰 감성 막대 그래프**: 전체 영화 간 감성 분포 비교

---

## 4. 문제 해결 및 주요 고려사항
- 더보기 버튼을 통해 영화 목록을 확장할 수 있었지만, 안정성을 위해 19개 현재 상영작만 대상으로 수집
- 리뷰가 한 영화당 60개 이상 존재하는 경우, 반복 페이지네이션을 통해 전량 수집하도록 처리
- Jupyter Notebook 환경에서 시각화 저장 및 결과 디렉토리 관리 자동화
